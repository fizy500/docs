Endpoint:

`Get www.themealdb.com/api/json/v1/1/search.php?s={name of meal} `

Replace name of meal with the meal of your choice

Request Example:

`Get https://www.themealdb.com/api/json/v1/1/search.php?s=Spicy%20Arrabiata%20Penne `
