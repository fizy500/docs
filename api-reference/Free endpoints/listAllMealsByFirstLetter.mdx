---
title: 'List all meals by first letter'
api: 'Get https://www.themealdb.com/api/json/v1/1/search.php?f=a'
description: "This returns all meals according to a particular letter"
---


Endpoint:

`https://www.themealdb.com/api/json/v1/1/search.php?f={first letter of meal}`

Replace first letter of meal with the letter of your choice

Request Example:

 `https://www.themealdb.com/api/json/v1/1/search.php?f=b`

<RequestExample>
```curl 
curl --location --request Get 'www.themealdb.com/api/json/v1/1/search.php?f=y'
```
</RequestExample>
